Configuración

Primero iremos a la pestaña "Boot" y veremos un orden de arranque preestablecido que podremos personalizar a nuestras necesidades, en este caso para personalizar un orden seguro(recordemos que las opciones de USB y red estarán desactivadas) elegiremos el siguiente orden:

  • Primero elegiremos el disco duro que tengamos instalado
  • De segunda opción un disquete USB(USB FDD)
  • A continuación un disco duro USB(USB HDD)
  • Luego un disco CD
  • Y como últimas opciones elegiremos las opciones de red (primero IPv4 y luego IPv6)

Configuraciones adicionales

  • Configurar una contraseña para todos los discos duros instalados
