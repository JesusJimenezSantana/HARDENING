Para configurar el arranque de la BIOS desde dispositivos externos primero nos iremos a la pestaña "Advanced" y seleccionaremos las opciones que necesitemos, en este caso para asegurar nuestra BIOS deshabilitaremos las opciones:

  • Wake up On LAN S5 Support
  • Network Stack

Configuraciones adicionales

Si queremos añadir algo más de seguridad a nuestra BIOS en ese caso podemos habilitar un par de configuraciones extra como:

  • Habilitar Intel® BIOS Guard Technology
    Intel BIOS Guard Technology es una opción de seguridad en las BIOS Intel a partir de Windows 8 que funciona igual que Secure Boot, por lo que con esto añadiremos una capa de seguridad extra en caso de que el Secure Boot falle.
    Para habilitarlo iremos a la pestaña "Advanced" y seleccionaremos "Intel® BIOS Guard Technology":

  • Habilitar Secure Boot
    Secure Boot es una opción de seguridad de la BIOS que impide que se ejecute software no firmado o certificado durante el boot.
